# 1.【入门】点阵字生成器

## 开始

### 在输入框中输入以下提示词：
```
请开发一个功能完整的点阵字生成器，要求至少包含以下核心功能：
笔形
1．基础参数配置：
涂鸦
﹣允许用户自定义点阵的行数和列数（范围建议5-32)
﹣提供字体大小调节功能（含预览效果）- 支持选择不同的字体样式（如宋体、黑体、颜色、马赛克等）
2．点阵生成与编辑：
．可通过输入文字自动生成对应点阵图案
橡皮擦
﹣提供手动编辑功能，允许用户点击调整单个像素点状态
粗细
．支持实时预览生成效果
```

```
实时预览功能删除，字体大小预览也删除，最终保留左侧配置功能，右侧编辑功能。编辑功能的行和列上需要标注数字。再添加一个把内容导出为图片的功能。导出的图片内容中也需要有标注的数字
```
